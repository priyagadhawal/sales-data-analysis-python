Sales Data Analysis using Python
Project Overview
This project performs exploratory data analysis (EDA) on a retail sales dataset to uncover sales trends, profitability, and category-wise performance.
Tools & Technologies
⦁	Python
⦁	Pandas
⦁	NumPy
⦁	Matplotlib
⦁	Jupyter Notebook
Key Steps Performed
⦁	Data loading and inspection
⦁	Data cleaning and preprocessing
⦁	Grouped analysis using Pandas
⦁	Visualization of sales trends
⦁	Business insight generation
Key Insights
⦁	Technology category contributes the highest revenue.
⦁	Furniture category has instances of negative profit.
⦁	North and East regions are more profitable compared to others.
Conclusion
The analysis helps businesses identify profitable regions and categories, enabling data-driven decision-making.
